# Contributing to AI Grain Counter

First off, thank you for considering contributing to AI Grain Counter! It's people like you that make this project a great tool for the agricultural community.

## How Can I Contribute?

### Reporting Bugs
- Use the [GitHub issue tracker](https://github.com/anandmahadev/grain.detector/issues) to report bugs.
- Describe the bug in detail and provide steps to reproduce.

### Suggesting Enhancements
- Enhancement suggestions are tracked as GitHub issues.
- Provide a clear and concise description of the suggestion.

### Pull Requests
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/amazing-feature`).
3. Make your changes.
4. Ensure your code follows the project's coding standards.
5. Commit your changes (`git commit -m 'Add some amazing feature'`).
6. Push to the branch (`git push origin feature/amazing-feature`).
7. Open a Pull Request.

## Coding Standards
- Use `black` for formatting.
- Follow `PEP 8` guidelines for Python code.
- Add docstrings to all new functions and classes.
